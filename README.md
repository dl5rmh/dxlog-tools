# DXLog.net Tools

[![Check C# Scripts](https://github.com/dl5rmh/dxlog-tools/actions/workflows/check.yml/badge.svg)](https://github.com/dl5rmh/dxlog-tools/actions/workflows/check.yml)

A collection of scripts, configurations, best practices, and documentation for [DXLog.net](https://dxlog.net/), the contest logging software.

## Contents

This repository gathers various customizations and resources for DXLog.net, including:

- C# scripts in folders by use case
- Documented keyboard shortcuts and macros with clear explanations

## Usage

Each script or configuration includes a short description of its purpose and, where applicable, the keyboard shortcut or text command used to invoke it. Scripts are written in C# and can be loaded via DXLog.net's **Tools → Scripts** menu.

Refer to the [DXLog.net Wiki](https://dxlog.net/docs/index.php?title=Main_Page) for general information on scripting and configuration.
