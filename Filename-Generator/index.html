DXLog.net Contest Filename Generator

Generates a contest log file name using the contest and Cabrillo names from the DXLog.net contest list.

Typical usage: create a folder using the generated name yourself. For a new file in DXLog, enter the name without a file extension.

Tip: bookmark this page with URL parameters to pre-fill fields, e.g. ?callsign=DL5RMH&mode=CW&year=2026&week=1&cabrillo=CQ-WW-CW. Supported parameters: callsign, mode, year, week, cabrillo (Cabrillo name, including {MODE} if applicable).
Contest list generated from dxlog.net/sw/contestlist.php on each deployment.
